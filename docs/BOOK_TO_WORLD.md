# BOOK → WORLD: Как книга становится городом

Кидаешь книгу — получаешь не идентичный, а духовно родной мир.

---

## 1. Жанр книги → Тип поселения

| Жанр | Что строится | Почему |
|------|-------------|--------|
| essay (эссе) | Деревня / Усадьба | 5-8 кварталов, низкая плотность, много воздуха |
| treatise (трактат) | Город / Столица | 10-20 кварталов, сетка, академические кварталы |
| dialogue (диалог) | Агора / Форум | Радиальный layout, центральная площадь, круговые дороги |
| poetry (поэзия) | Сад / Парк | Organic layout, фонтаны, беседки, мало зданий |
| manifesto (манифест) | Крепость / Цитадель | Brutalist, стены, флаги, площадь митингов |
| autobiography (автобиография) | Усадьба с пристройками | Deformed layout, разные эпохи рядом, пристройки разных стилей |
| critique (критика) | Руины + новое строительство | Два слоя: старые развалины + современные здания между ними |

| Экзистенциализм | organic | wood | Волны, кривые линии, лианы, природа внутри |

| Аналитическая фил. | minimal | glass | Прямые линии, отсутствие декора, прозрачность |
| Прагматизм | brick | brick | Ремесленность, функция, земля, практичность |

## 6. Конкретные примеры

### Гессен — «Основы педагогики» (1925)
- Жанр: treatise → Город-академия
- Школа: неокантианство → classical + stone
- Эмоция: сдержанный академизм с тёплой заботой → dusk, dust, bells
- Структура: 3 части → 3 района (Нравственный, Научный, Национальный)
- Плотность: 8/10 → 14 кварталов, dense
- Мир: Каменные здания с колоннами, широкие улицы, библиотеки, фонтаны, колокольный звон, редкие прохожие в плащах, туман

### Платон — «Государство» (-380)
- Жанр: dialogue → Агора с радиальными дорогами
- Школа: классицизм → classical + marble
- Эмоция: ясность, просветлённость → day, silence, white marble
- Мир: Совершенная симметрия, 10 секторов, центральная площадь с фонтаном, колоннады, белые стены, редкие философы в тогах

### Ницше — «Так говорил Заратустра» (1883)
- Жанр: poetry + manifesto → Крепость на горе
- Школа: немецкий идеализм → baroque + metal
- Эмоция: дионисийское vs аполлонское → midnight, embers, red vs green
- Мир: Горы, обрывы, крепость с шпилем, огонь на башнях, орлы кружат, пустыня внизу, ветер, грохот

### Кафка — «Процесс» (1925)
- Жанр: novel (не в schema, но deformed) → Лабиринт-бюро
- Школа: постмодерн → deconstructivist + concrete
- Эмоция: тревога, бюрократия → midnight, machinery, fog 0.8
- Мир: Бесконечные коридоры, одинаковые здания, узкие окна, лифты, туман, машинный гул, отсутствие людей

### Маркс — «Капитал» (1867)
- Жанр: treatise → Индустриальный город
- Школа: марксизм → constructivist + concrete + metal
- Эмоция: классовая борьба → night, sparks, dead vegetation
- Мир: Фабрики с трубами, жилые бараки, заводы, мосты, краны, красные флаги, дым, рабочие в рядах

## 7. Инфраструктура — что в каждом мире

Каждый мир получает набор инфраструктуры по умолчанию + дополнительно по жанру:

### Базовый набор (всегда)
- Street lamps (фонари) — вдоль дорог
- Benches (скамейки) — в парках
- Signposts (указатели) — на перекрёстках
- Pavement (тротуары) — вдоль зданий

### Дополнительно по жанру
| Жанр | Дополнительная инфраструктура |
|------|------------------------------|
| essay | Колодцы, заборы, огороды, пчелиные ульи |
| treatise | Университеты, библиотеки, обсерватории, мемориалы |
| dialogue | Ростры, трибуны, сцены, кафе, фонтаны |
| poetry | Скульптуры, беседки, арки цветов, мостки через пруды |
| manifesto | Баррикады, трибуны, флаги, громкоговорители, плакаты |
| autobiography | Фамильные гербы, портреты, детские площадки, кладбище |
| critique | Строительные леса, краны, развалины, заборы «осторожно» |

## 9. Ассеты для скачивания

### Kenney (уже скачано)
- `building-small-[a-d].glb` — 4 типа зданий
- `building-garage.glb` — гараж/фабрика
- `grass.glb` — лужайка
- `grass-trees.glb`, `grass-trees-tall.glb` — деревья с травой
- `pavement.glb` — тротуар
- `pavement-fountain.glb` — фонтан
- `road-*.glb` — дороги (straight, corner, intersection, split, lightposts)

### Дополнительные источники (ручная загрузка)

**Quaternius:** https://quaternius.com
- Animals pack — олень, кролик, птицы
- Nature pack — разные деревья, камни, цветы

**Poly Pizza:** https://poly.pizza
- Low poly buildings
- Fantasy structures

**Sketchfab CC0:** https://sketchfab.com/search?features=downloadable&sort_by=-likeCount&type=models
- Поиск: "low poly building", "low poly tree", "low poly animal"

**Godot Asset Library:** https://godotengine.org/asset-library
- Поиск: "city", "nature", "lowpoly"

### Куда класть
```
assets/
  models/
    buildings/      ← Kenney здания
    vegetation/     ← деревья, кусты, цветы (скачать дополнительно)
    animals/        ← животные (скачать с quaternius.com)
    infrastructure/ ← фонари, скамейки, мосты, фонтаны
    terrain/        ← камни, холмы, пещеры
    water/          ← мосты, пристани, лодки
  textures/
    colormap.png    ← Kenney texture atlas
  materials/
    building_mat.tres
    water_mat.tres
    terrain_mat.tres
```

## 10. Параметры → Параметры города (быстрая шпаргалка)

| JSON param | Что строится | Godot код |
|-----------|-------------|-----------|
| gridSize | N×N кварталов | `for x in gs: for z in gs:` |
| blockSize | Размер клетки (м) | `cell_size = Vector3(bs, 0, bs)` |
| roadWidth | Ширина дороги | `road_mesh.size.x = roadWidth` |
| layoutType | Паттерн дорог | `match layoutType` |
| connectivity | Доп. переулки | `if randf() < connectivity:` |
| terrainRoughness | Высота холмов | `height *= roughness` |
| waterPresence | Количество рек/озёр | `river_count = wp * 5 + 1` |
| buildingDensity | Застроенность | `building_cells = total * density` |
| parkRatio | Парки | `park_cells = total * parkRatio` |
| style | Форма зданий | `match style` → меш |
| material | Цвет фасада | `Color.from_hsv(hue, sat, bright)` |
| skyscraperRatio | Высотные здания | `if randf() < skyscraperRatio: tall` |
| blockSubdivisions | Зданий в квартале | `_subdivide_cell(cell, count)` |
| dominantHue | Цвет города | `Color.from_hsv(hue/360, ...)` |
| timeOfDay | Освещение | `sun.energy = energy_by_tod[tod]` |
| fogDensity | Туман | `env.fog_density = fog * 0.008` |
| windSpeed | Движение частиц | `particles.gravity.x = windSpeed` |
| particleType | Тип частиц | `match particleType` → цвет |
| particleCount | Количество частиц | `particles.amount = count` |
| treeDensity | Количество деревьев | `tree_count = cells * density * 1.5` |
| vegetationType | Тип деревьев | `match vegType` → меш |
| windowLightFrequency | Горящие окна | `if randf() < frequency: lit` |
